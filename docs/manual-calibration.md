# Ручная калибровка и отладка PTZ Pano

Механизм предназначен для устранения погрешностей наведения камеры на точки панорамы путем создания "карты поправок" на основе реальных измерений.

## Основные возможности

1.  **Интерполяция поправок**: Система использует метод взвешенного среднего (на основе расстояния до опорных точек) для расчета коррекции координат Pan, Tilt и Zoom в любой точке панорамы.
2.  **Привязка к центру**: Оператор может совместить объект на панораме с центральным перекрестием, навести камеру на этот же объект вручную и нажать "Привязать к центру".
3.  **Коррекция Zoom**: Система учитывает разницу между расчетным и реальным зумом, позволяя более точно попадать в нужный угол обзора.
4.  **Визуализация**: Все точки калибровки отображаются в виде зеленых маркеров на панораме (в режиме сетки).
5.  **Статистика в реальном времени**: В режиме калибровки отображаются средние коэффициенты `Pan Units / Degree` и `Tilt Units / Degree`.

## Процесс калибровки

1.  Запустите просмотрщик и включите режим **"Сетка"**.
2.  Найдите на панораме характерный объект и совместите его с центральным перекрестием (reticle).
3.  Используя пульт или команды управления, наведите саму камеру на этот же объект (совместите центр кадра с объектом). При необходимости подстройте зум.
4.  Нажмите кнопку **"Привязать к центру"**. Появится подтверждение и на панораме отобразится зеленая точка.
5.  Повторите процедуру для нескольких зон панорамы (рекомендуется от 4 до 9 точек в разных частях).
6.  После этого любое наведение через "Навести камеру" или клик по панораме будет автоматически учитывать накопленные поправки.

## Технические детали

- **Хранение**: Точки сохраняются в `data/calibration/corrections.json`.
- **Алгоритм**: Для расчета поправки в новой точке `(Y, P)` берется сумма ошибок во всех опорных точках, взвешенная по `1 / (dist + 0.1)`. 
- **Сброс**: Кнопка "Очистить" удаляет файл коррекций и возвращает систему к стандартным коэффициентам из конфигурации.
